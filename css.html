QUE ES CSS

+

Es el lenguaje utilizado para describir la presentación de documentos HTML o XML. CSS describe como debe ser renderizado el elemento estructurado en pantalla.

CSS es uno de los lenguajes base de la Open Web y posee una especificación estandarizada por parte del W3C. Desarrollado en niveles, CSS1 es ahora obsoleto, CSS2.1 es una recomendación y CSS3, ahora dividido en módulos más pequeños, está progresando en camino al estándar.

LIMITACIONES

+
  • Los selectores no pueden ascender
  • No se pueden nombrar estilos CSS
  • Las pseudo-clases dinámicas no se pueden controlar
  • No se pueden incluir estilos de una regla dentro de otra
  • No se puede seleccionar texto específico sin alterar el marcado

POSICIONAMIENTO

+

El modelo de caja CSS es esencialmente una caja que envuelve cada elemento HTML. Consiste en: márgenes, bordes, relleno y el contenido

imagen
  • Propiedad static

    • Los elementos posicionados estáticos no se ven afectados por las propiedades superior, inferior, izquierda y derecha

  • Propiedad relative

    • Con tan solo establecer la propiedades superior, de un elemento relativamente posicionado hará que se ajuste fuera de su posición normal

  • Propiedad fixed

    • se coloca en relación con la ventana gráfica, lo que significa que siempre permanece en el mismo lugar, incluso si se desplaza la página

  • Propiedad absolute

    • Se coloca en relación con el contenedor posicionado más cercano

MEDIAS QUERIS

+

La regla @media, introducida en CSS2, es una técnica popular para entregar una hoja de estilo personalizada a equipos de escritorio, portátiles, tabletas y teléfonos móviles (como teléfonos iPhone y Android)

PSEUDOCLASES

+

Una pseudoclase CSS es una palabra clave que se añade a los selectores y que especifica un estado especial del elemento seleccionado.

  • :hover

    • aplicará un estilo cuando el usuario pase el cursor sobre el elemento especificado por el selector.

  • :activate

    • especifica y selecciona un elemento activo, y se utiliza normalmente para enlaces < a > pero puede ser utilizada para otros elementos

  • :link

    • se utiliza para seleccionar y dar estilo a los enlaces que no fueron visitados
